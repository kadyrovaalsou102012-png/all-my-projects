# Мои проектные работы по Python и Django

В этом репозитории собраны 4 финальных проекта, выполненных в рамках обучающего курса.

## Список проектов:

### 1. Игра «Камень, ножницы, бумага» (game.py)
* Описание: Консольная игра против компьютера. 
* Функционал: Игра ведется до 3 победных очков одного из игроков, ведется подсчет раундов и автоматический вывод итогового счета.

### 2. Приложение «Викторина и заметки» (quiz_and_notes.py)
* Описание: Полноценная консольная программа с системой авторизации.
* Функционал: * Регистрация и вход пользователей (с валидацией пароля).
  * Создание карточек с вопросами и ответами, запуск викторины с таймером на 30 секунд.
  * Личный блокнот: добавление, просмотр и удаление заметок.

### 3. Телеграм-бот (telegram_bot.txt)
* Описание: Ссылка на работающего интерактивного бота в Telegram.
* Функционал: Бот запущен на сервере и доступен для тестирования в реальном времени.

### 4. Веб-приложение «Блог» на Django (django_blog.py)
* Описание: Бэкенд-логика для полноценного сайта-блога.
* Функционал: * Созданы модели для статей (Article) и комментариев (Comment).
  * Реализована регистрация моделей в панели администратора (admin.py).
  * Настроена система маршрутов (urls.py) и обработка запросов (views.py) для просмотра, создания, редактирования и удаления статей, а также добавления комментариев в хронологическом порядке.
